- react, react router

- zustand

- socket.io
- react hook form, zod

## How to run the app

1. Clone this repository

```cmd
git clone https://github.com/anastshak/Stack-Overflow-analogue.git
```

1. Install dependencies

```cmd
npm install
```

3. Start

```cmd
npm run dev
```

## Useful scripts

1. For identifying coding style issues and potential errors:

```cmd
npm run lint
```

1. For fixing errors

```cmd
npm run lint:fix
```

3. For checking coding style issues:

```cmd
npm run prettier
```

4. For code formatting and ensuring uniform code style:

```cmd
npm run format:fix
```

5. For testing:

```cmd
npm run test
```
